# Обработка PDF платежных поручений

## Цель

Сделать локальную Windows-программу, которая читает папку с PDF платежных поручений, извлекает данные без OCR и формирует таблицу результата для режимов `ПСК` и `ИС`.

## Текущий статус

- PDF читаются через `pypdf`; OCR не используется.
- Поддержаны разные банковские шаблоны: Сбербанк, Альфа, Точка, ИП Мочалов, Инвестстрой.
- Результат пишется в один Excel-файл с листами по режимам.
- Есть встроенная таблица проверки и редактирования перед сохранением.

## Важные решения

- Определять шаблон не по имени файла, а по текстовым признакам.
- Многострочные поля собирать по маркерам, а не по номерам строк.
- Если правило классификации не найдено, поле остается пустым.
- ПП считаются единственным типом файлов, который должен попадать в папки `_ПП/YYYY.MM.DD`.

## Проверки

- Все исторические PDF должны читаться без ошибок.
- Один PDF должен давать одну строку результата.
- Повторный запуск не должен создавать ежедневные копии Excel.
