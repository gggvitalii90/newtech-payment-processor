# Вопросы для пополнения словаря

Этот файл нужен, чтобы постепенно пополнять `dictionaries.json`. Можно отвечать частями.

## Объекты

Для каждого объекта нужны варианты, как сотрудники пишут его в MAX, и имя папки на Google Drive.

Формат ответа:

```text
ПСК Ньютек:
- пишут: пск, офис, ньютек
- папка Drive: 01_ ПСК Ньютек

Производство:
- пишут: производство, прод, цех
- папка Drive: 02_ Производство
```

## Ответственные

Формат ответа:

```text
Родин.К:
- пишут: родин, кирилл, родин к

Раздрогина.С:
- пишут: раздрогина, светлана, раздрогина с
```

## Проекты и статьи

Формат ответа:

```text
Офис:
- пишут: офис

Расходники:
- пишут: расходники, материалы, мелочевка
```

## Папки месяцев

Если внутри объектов месяцы называются нестандартно, присылай примеры:

```text
2026-05 -> 05 май
2026-06 -> Июнь 2026
```

Пока программа не создает новые папки месяцев автоматически: если путь не найден, строка получает `Статус разбора = Нужно разобрать`.
